Device Fingerprinting

Utilitar client-side care genereaza un hash unic pe baza caracteristicilor browser-ului si device-ului.

  • Screen: Rezolutie, color depth, pixel ratio
  • WebGL: GPU vendor & renderer
  • Canvas: Rendering differences per device
  • Navigator: Language, platform, cores, RAM
  • Fonts: Font width probe (8 test fonts)
  • Touch: Touch support, max touch points
  • Hash: SHA-256 via SubtleCrypto
UTILITARNOU
📡

API: /api/fraud/track

Endpoint server-side care inregistreaza fingerprint + IP la fiecare login/register.

  • IP tracking: x-forwarded-for, x-real-ip, socket IP
  • Auto-flag: Detectie automata daca acelasi fingerprint/IP apare la alt cont
  • Severity: low (1 link), medium (2), high (3+)
  • Cross-ref: Flag-uri create bidirectional intre conturile legate
  • Fire-and-forget: Nu blocheaza flow-ul de autentificare

  • Scan window: Ultimele 24h de cereri
  • Output: Flag-uri create in colectia fraudFlags
